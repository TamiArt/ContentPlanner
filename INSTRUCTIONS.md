# Content Planner AI - Инструкция для пользователя

## Быстрый старт

1. **Первый запуск**
   - При первом запуске откроется экран приветствия
   - Выберите "Загрузить демо-данные" для примеров или "Начать работу" для чистого старта

2. **Установка как приложение (PWA)**
   - **iOS**: Safari → Поделиться → На экран Домой
   - **Android**: Chrome → Меню → Добавить на главный экран
   - **Desktop**: Chrome/Edge → значок установки в адресной строке

## Основной рабочий процесс

### 1. Создание идеи
- Нажмите "Добавить идею" на главном экране
- Введите текст идеи (минимум)
- Опционально: добавьте платформу, формат, нишу

### 2. Превращение идеи в пост
- Откройте "Банк идей"
- Найдите нужную идею
- Нажмите "Создать пост"

### 3. Работа с постом

**Заполнение карточки поста:**
- **Название** - краткое название для себя
- **Тема** - о чём пост
- **Ниша** - ваша ниша (искусство, обучение, рецепты)
- **Рубрика** - тип контента (экспертный, личный, обучающий)
- **Платформа** - где публикуете (TikTok, Instagram, etc.)
- **Формат** - карусель, видео, Reels, пост
- **Цель** - подписки, охваты, сохранения, комментарии
- **Количество слайдов** - для каруселей
- **Стиль** - как подавать (экспертно, живо, дружелюбно)

**Генерация структуры:**
1. Заполните основные поля
2. Нажмите "Сгенерировать структуру"
3. Получите шаблон структуры под ваш формат

**Получение текста от нейросети:**
1. Заполните все поля карточки
2. Нажмите "Скопировать промпт"
3. Откройте ChatGPT, Claude или другую нейросеть
4. Вставьте промпт
5. Получите готовый текст
6. Скопируйте текст обратно в поле "Текст поста"

**Сохранение:**
- Нажмите "Сохранить"
- Пост обновится в Kanban-доске

### 4. Управление статусами

Перетаскивайте карточки между столбцами:
- **Идея** - только родилась мысль
- **В работе** - начали разработку
- **Нужен текст** - структура готова, нужен текст
- **Нужен визуал** - текст готов, нужны фото/видео
- **Готово** - всё готово к публикации
- **Запланировано** - установлена дата публикации
- **Опубликовано** - пост опубликован
- **Нужно проанализировать** - добавить аналитику

### 5. Планирование публикации
- Установите дату и время публикации
- Пост появится в разделе "Сегодня" в нужный день
- Переместите в "Опубликовано" после публикации

### 6. Добавление аналитики
После публикации заполните:
- Ссылка на пост
- Просмотры
- Лайки
- Комментарии
- Сохранения
- Репосты
- Новые подписчики
- Выводы

## Работа с промптами

### Переменные в шаблоне промпта
Доступные переменные:
- `{ниша}`
- `{рубрика}`
- `{тема}`
- `{платформа}`
- `{формат}`
- `{цель}`
- `{аудитория}`
- `{стиль}`
- `{тон}`
- `{количество_слайдов}`
- `{ключевые_слова}`
- `{описание_идеи}`
- `{визуальные_заметки}`
- `{ограничения}`
- `{язык}`

### Редактирование шаблона промпта
1. Откройте "Настройки"
2. Найдите "Шаблон промпта для нейросети"
3. Редактируйте текст
4. Используйте переменные в фигурных скобках
5. Нажмите "Сохранить шаблон"

## Резервное копирование

### Экспорт данных
1. Откройте "Настройки"
2. Нажмите "Экспортировать все данные в JSON"
3. Сохраните файл на устройство

### Импорт данных
1. Откройте "Настройки"
2. Нажмите "Импортировать данные из JSON"
3. Выберите ранее сохранённый файл

**Важно:** Регулярно делайте резервные копии! Данные хранятся только в браузере.

## Советы по использованию

### Для максимальной эффективности:
1. **Создавайте идеи сразу** - как только придёт в голову
2. **Планируйте заранее** - заполняйте календарь на неделю вперёд
3. **Используйте шаблоны** - они ускоряют создание контента
4. **Анализируйте результаты** - записывайте статистику каждого поста
5. **Делайте бэкапы** - экспортируйте данные раз в неделю

### Оптимизация промптов:
- Заполняйте максимум полей для лучшего результата
- Чем детальнее описание, тем точнее ответ нейросети
- Экспериментируйте со стилями и тоном
- Сохраняйте успешные формулировки в заметках

### Работа с форматами:
- **Карусель** - укажите количество слайдов (обычно 5-10)
- **Reels/TikTok** - опишите визуал подробно
- **Сторис** - разбейте на последовательность
- **Пост** - укажите желаемую длину

## Частые вопросы

**Q: Данные пропали после очистки браузера**
A: Данные хранятся локально в браузере. Регулярно делайте экспорт в JSON.

**Q: Промпт не подставляет значения**
A: Проверьте, что переменные написаны в точности как в списке, в фигурных скобках.

**Q: Как перенести данные на другое устройство?**
A: Экспортируйте данные в JSON на первом устройстве, затем импортируйте на втором.

**Q: Приложение работает без интернета?**
A: Да, после первой загрузки. Но для копирования промптов в нейросети нужен интернет.

**Q: Можно ли работать с командой?**
A: Пока нет функции синхронизации. Используйте экспорт/импорт JSON для обмена данными.

## Техническая поддержка

Приложение полностью бесплатное и работает локально. Все данные принадлежат только вам.

При возникновении проблем:
1. Попробуйте обновить страницу
2. Проверьте версию браузера (рекомендуется последняя)
3. Сделайте экспорт данных
4. Очистите кэш браузера и импортируйте данные обратно
