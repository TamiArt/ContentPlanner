# Content Planner AI

Бесплатное PWA-приложение для планирования контента в соцсетях без использования платных API.

## Особенности

- ✅ **Полностью бесплатно** - без платных API и внешних сервисов
- 💾 **Локальное хранение** - все данные сохраняются в браузере
- 📱 **PWA** - работает как приложение на iOS и Android
- 🎯 **Kanban-доска** - управление постами с drag & drop
- 💡 **Банк идей** - быстрое сохранение идей для контента
- 🤖 **Генератор промптов** - автоматическое создание промптов для ChatGPT и других нейросетей
- 📅 **Календарь публикаций** - планирование контента
- 📊 **Ручная аналитика** - отслеживание результатов
- 🎨 **Шаблоны постов** - готовые структуры для разных форматов

## Как это работает

1. **Создайте идею** - быстро сохраните идею для будущего поста
2. **Создайте пост** - заполните детали: платформа, формат, цель, тема
3. **Скопируйте промпт** - приложение автоматически сгенерирует промпт для нейросети
4. **Получите текст** - вставьте промпт в ChatGPT или другую нейросеть
5. **Сохраните результат** - вставьте готовый текст обратно в приложение
6. **Запланируйте публикацию** - установите дату и время
7. **Опубликуйте** - используйте готовый контент
8. **Анализируйте** - вручную добавьте статистику после публикации

## Установка как PWA

### На iOS (iPhone/iPad):
1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Выберите "На экран Домой"
4. Нажмите "Добавить"

### На Android:
1. Откройте приложение в Chrome
2. Нажмите меню (три точки)
3. Выберите "Добавить на главный экран"
4. Нажмите "Добавить"

### На Desktop (Chrome/Edge):
1. Откройте приложение в Chrome или Edge
2. Справа в адресной строке появится значок установки
3. Нажмите на него и выберите "Установить"

## Переменные для промпта

Приложение поддерживает следующие переменные в шаблоне промпта:

- `{ниша}` - ниша контента
- `{рубрика}` - рубрика поста
- `{тема}` - тема поста
- `{платформа}` - социальная сеть
- `{формат}` - формат контента (карусель, видео, пост)
- `{цель}` - цель публикации (подписки, сохранения, комментарии)
- `{аудитория}` - целевая аудитория
- `{стиль}` - стиль подачи
- `{тон}` - тон коммуникации
- `{количество_слайдов}` - количество слайдов для карусели
- `{ключевые_слова}` - ключевые слова
- `{описание_идеи}` - описание идеи поста
- `{визуальные_заметки}` - заметки о визуале
- `{ограничения}` - ограничения
- `{язык}` - язык ответа

## Экспорт и резервное копирование

Вы можете экспортировать все данные в JSON файл через раздел "Настройки":
- Все посты
- Все идеи
- Все шаблоны
- Настройки приложения

Рекомендуется периодически создавать резервные копии данных.

## Безопасность данных

Все данные хранятся локально в браузере пользователя. Приложение не отправляет данные на внешние серверы.

**Важно**: При очистке данных браузера все посты и идеи будут удалены. Регулярно делайте резервные копии через экспорт в JSON.
