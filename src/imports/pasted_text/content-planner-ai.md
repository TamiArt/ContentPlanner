Приложение должно работать как PWA: в браузере, на iOS и Android через “Добавить на главный экран”.

- Tailwind CSS
- IndexedDB или localStorage для хранения данных
- PWA manifest + service worker
- Без backend
- Без платных API
- Без авторизации на первом этапе

Название приложения:
Content Planner AI

Главная идея:
Content Planner AI — это бесплатный локальный помощник для планирования контента, который помогает пользователю системно создавать идеи, тексты, сценарии, визуальные задания и распределять контент по календарю публикаций.

Приложение не генерирует тексты через API, но использует:
1. Готовые шаблоны структур постов.
2. Генератор структуры поста без AI.
3. Генератор универсального промпта для ChatGPT или любой другой нейросети.
4. Кнопку “Скопировать промпт”.
5. Ручное сохранение готового текста обратно в карточку поста.

Главная задача пользователя:
Я открываю приложение → вижу, что мне сегодня нужно сделать → получаю идею, структуру поста, промпт для нейросети, поле для текста, хэштеги, задачу для визуала → ставлю статус → публикация попадает в календарь → после публикации вношу аналитику.

Приложение должно решать проблему несистемного создания контента.

--------------------------------------------------
18. ИТОГОВЫЙ ПОЛЬЗОВАТЕЛЬСКИЙ СЦЕНАРИЙ
--------------------------------------------------

Пользователь открывает приложение.

На экране “Сегодня” видит:
- что нужно опубликовать;
- что нужно дописать;
- что нужно доснять;
- какие посты готовы;
- какие посты нужно проанализировать.

Пользователь нажимает “Быстро добавить идею”.
Вводит:
“Я нарисовала интерьерный скетч и вписала туда свою реальную картину.”

Потом выбирает:
Платформа: TikTok
Формат: карусель
Цель: подписки
Стиль: экспертно, но живо

Приложение создаёт карточку поста.

Пользователь нажимает:
“Сгенерировать структуру”

Приложение выдаёт:
1. Хук
2. Проблема
3. Мини-объяснение
4. Пример
5. Призыв подписаться

Пользователь нажимает:
“Скопировать промпт для ChatGPT”

Приложение собирает промпт из полей карточки.

Пользователь вставляет промпт в ChatGPT, получает текст, возвращается в приложение и сохраняет текст в карточку.

Потом ставит статус:
“Готово”

Добавляет дату публикации.

Пост появляется в календаре.

После публикации пользователь ставит статус:
“Опубликовано”

Затем вручную добавляет аналитику:
просмотры, лайки, сохранения, комментарии, подписки.

Приложение помогает понять, какие форматы работают лучше.

Сделай приложение аккуратным, удобным, быстрым и понятным. Главный фокус — системность создания контента без платных API.
